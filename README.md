# Healthcare Data Pipeline (SQL Server)

This project simulates a healthcare data migration and processing system inspired by Epic-style EHR workflows.

## Features

- Patient, Visit, and Lab data modeling
- Data ingestion simulation
- SQL-based transformations
- Performance optimization (planned)

## Tech Stack

- SQL Server
- T-SQL
- SSMS

## Project Goal

To simulate real-world healthcare data pipelines and strengthen SQL + Data Engineering fundamentals.
